Down To Code

Matthew loves crafting new things. Sparking new ideas. Creating new innovations. Empowering people's lives.


  • Plan

    March 2016 - present

    Remember paper planners? Interweaving digital pens to enable paper-like scheduling on tablets.


    Crafted on C# and XAML, with APIs for UWP, Surface Pen, and Cortana. Sparked in BrickHack 2. (Hacky!) Source code available here on Github.

  • DQGen-Swahili

    February 2016 - present

    Automating generation of multiple choice questions in Swahili for reading comprehension assessment.


    Made with care on Java with PDFbox. One of the compnents to CMU's Global Learning XPRIZE Project. Source code available on Github, here.

  • Daylight

    February 2016

    Enabling internet-free crowdsourced information sharing through SMS for any application. Prototype API.


    Built on Java with APIs for Android and Twilio. Created in TartanHacks 2016. Shared source code available on Github.

  • Dawn

    Fall 2014

    A developer's calendar: applying NLU to enable keyboard-only event scheduling: “eat pie at 4p tmrw”.


    First project: Created on Python with regex and tkinter libraries. Made for 15-112 Term Project. Source code available here.
